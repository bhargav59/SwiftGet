SwiftGet Options

Configure how the SwiftGet extension intercepts downloads and detects video streams.

General

Automatically route qualifying downloads to SwiftGet.

Injects a "Download with SwiftGet" button on YouTube, Vimeo, etc.

File Type Rules

Comma-separated list of extensions to route through SwiftGet. Leave blank to use defaults.

Domain Rules

If set, only downloads from these domains are intercepted. One per line or comma-separated.

Downloads from these domains are never sent to SwiftGet.
